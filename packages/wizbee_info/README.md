# wizbee_info

Landing page for **WizBee** — the integrated LLM agent of [btree.at](https://www.btree.at).

Live at: [wiz-bee.com](https://wiz-bee.com)

[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC%20BY--NC%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

WizBee is a conversational AI assistant built directly into b.tree, the beekeeping management platform. It lets professional beekeepers manage hives, treatments, harvests, and more through natural language, powered by Mistral AI.

## Stack

- [Astro](https://astro.build) (static output, minimal dependencies)
- No frameworks, no Tailwind — plain HTML, CSS, and Astro image optimisation

## Commands

| Command           | Action                                       |
| :---------------- | :------------------------------------------- |
| `npm install`     | Install dependencies                         |
| `npm run dev`     | Start local dev server at `localhost:4321`   |
| `npm run build`   | Build static site to `./dist/`               |
| `npm run preview` | Preview production build locally             |
